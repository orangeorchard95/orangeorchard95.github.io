我要在这里记录量化知识的学习内容。

这是「量化学习路径」系列的第〇篇。与其说是教程，不如说是一份公开的学习日志：我从零开始，把学到的东西整理成笔记发在这里，一方面逼自己把概念真正想清楚（讲不明白就是没学会），另一方面留下一条可以回溯的路径。

## 我打算怎么学

目前的路径规划（会随着学习不断修正）：

1. **基础概念** — 收益率、波动率、夏普比率这些最基本的语言
2. **数据处理** — 用 Python 获取和清洗行情数据
3. **回测框架** — 理解回测的原理和常见的坑（未来函数、幸存者偏差）
4. **经典策略** — 从最简单的均线策略开始，理解每类策略背后的逻辑
5. **风险管理** — 仓位、回撤控制
